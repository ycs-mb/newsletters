Tip of the Day

Actionable

Use Flash for coding, Pro for reasoning. Gemini 3 Flash beats Pro on SWE-bench (78% vs 76.2%) at $0.50/M vs $2-4/M input tokens. For coding tasks, always start with Flash — better results at 1/4 to 1/8 the cost. Reserve Pro for complex multi-modal reasoning across documents, not code.

# Use Flash for coding tasks (cheaper + better) gemini --model gemini-3-flash "refactor this function" < src/utils.py # Use Pro only for complex multi-source analysis gemini --model gemini-3.1-pro "analyze architecture docs" < docs/ # Use Flash-Lite for high-volume, cost-sensitive traffic curl -X POST "https://generativelanguage.googleapis.com/v1/models/gemini-3.1-flash-lite:generateContent"
