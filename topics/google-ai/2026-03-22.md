## Section 05: Developer Tip

Gemini 3 Flash beats Gemini 3 Pro on SWE-bench (78% vs 76.2%) at $0.50/M vs $2-4/M input tokens. For coding tasks, always start with Flash — you get better results at 1/4 to 1/8 the cost. Reserve Pro for complex multi-modal reasoning across documents, not code.

```bash
# Gemini CLI — use Flash for coding tasks
gemini --model gemini-3-flash "refactor this function for readability" < src/utils.py

# Use Pro only for complex multi-source analysis
gemini --model gemini-3.1-pro "analyze these architecture docs and propose migration plan" < docs/
```
