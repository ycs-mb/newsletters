Tip of the Day

Zero-Noise Extrapolation (ZNE) in Qiskit Runtime — One Flag That Buys You 3–5× Accuracy

When running variational algorithms (VQE, QAOA) on real hardware, enable ZNE via Qiskit's EstimatorV2 options without touching your circuit:

from qiskit_ibm_runtime import EstimatorV2
from qiskit_ibm_runtime.options import EstimatorOptions

options = EstimatorOptions()
options.resilience_level = 2              # Activates ZNE
options.resilience.zne_extrapolator = "exponential"
options.resilience.zne_noise_factors = [1, 3, 5]   # Must be odd integers

estimator = EstimatorV2(backend=backend, options=options)

resilience_level = 2 activates ZNE automatically. The noise_factors list scales gate error rates across three amplification levels; odd integers are required. Use exponential extrapolation for superconducting noise profiles, linear for trapped-ion systems. On 50–100 qubit circuits, this alone typically reduces systematic error by 3–5× with no changes to the circuit itself.
