Tip of the Day

Actionable

Use --bare for clean scripted -p calls — new in v2.1.81

When automating Claude Code with -p, the --bare flag strips all interactive machinery: hooks, LSP, plugin sync, and skill directory walks are skipped entirely. Output is clean and predictable — ideal for CI pipelines, cron jobs, and shell scripts.

ANTHROPIC_API_KEY=sk-ant-... claude -p "summarize changes in this PR" --bare < diff.txt

Two caveats: --bare disables OAuth and keychain auth entirely — you must supply ANTHROPIC_API_KEY explicitly. Auto-memory is also fully disabled in bare mode, so nothing is persisted across runs.
