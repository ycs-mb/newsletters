Tip of the Day

Actionable

Use --bare for CI/CD pipelines and scripted automation.

New in v2.1.81, the --bare flag strips all interactive overhead: hooks, LSP, plugin sync, skill directory walks, and auto-memory. It requires ANTHROPIC_API_KEY (OAuth and keychain auth are disabled in this mode). Perfect for cron jobs, build pipelines, or any context where Claude is invoked as a subprocess.

ANTHROPIC_API_KEY=sk-ant-... claude --bare -p "Summarize the diff" < git.diff

Benefits: ~80 MB less startup memory, no hook side-effects, deterministic I/O. Pair with --output-format json for machine-readable pipeline output. Note: auto-memory is fully disabled in --bare mode, so session context won't persist.
