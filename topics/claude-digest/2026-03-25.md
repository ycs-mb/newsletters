## Section 05: Tip of the Day

**Use `--bare` for clean scripted calls (new in v2.1.81)**

When scripting Claude Code with `-p`, the `--bare` flag strips everything non-essential: hooks, LSP, plugin sync, and skill directory walks are all skipped. Great for CI pipelines, cron jobs, or automation where you want raw output without interactive machinery.

```bash
ANTHROPIC_API_KEY=sk-... claude -p "summarize changes in this PR" --bare < diff.txt
```

Note: `--bare` disables OAuth/keychain auth — you must provide `ANTHROPIC_API_KEY` directly. Auto-memory is fully disabled in this mode.
