## Section 05: Tip of the Day

**Use `--bare` for CI/CD pipelines and scripted automation.**

New in v2.1.81, the `--bare` flag strips all interactive overhead: hooks, LSP, plugin sync, skill directory walks, and auto-memory. It requires `ANTHROPIC_API_KEY` (OAuth and keychain auth are disabled). Ideal for cron jobs, build pipelines, or wrapping Claude in another script.

```bash
ANTHROPIC_API_KEY=sk-... claude --bare -p "Summarize the diff" < git.diff
```

Benefits: ~80 MB less startup memory, no hook side-effects, predictable pure I/O. Pair with `--output-format json` for machine-readable output in pipelines.
