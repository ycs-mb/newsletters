## 💡 TIP OF THE DAY

Use the new `--bare` flag in Claude Code for scripted pipelines:
```
claude --bare --print "refactor this function" < myfile.py
```
`--bare` suppresses all UI chrome and outputs only the response — perfect for piping into other tools or CI scripts.
